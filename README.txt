ПОРТФОЛИО АННЫ СОКОЛОВОЙ

СТРУКТУРА:
- index.html — разметка сайта
- css/style.css — стили
- js/script.js — меню, анимации и текущий год
- assets/img/anna-portrait.png — фотография
- assets/icons — папка для будущих иконок

КАК ОТКРЫТЬ:
1. Откройте папку anna-portfolio-clean в VS Code.
2. Нажмите правой кнопкой по index.html.
3. Выберите Open with Live Server.

ВАЖНО:
Не переносите файлы по отдельности — пути между ними уже настроены.
